## How to Deploy on GitHub Pages

1. **Create a GitHub Repository**
   - Go to github.com and create a new repository named `pheliridge.github.io`

2. **Upload Files**
   - Upload `index.html`, `styles.css`, and `script.js` to the repository root

3. **Enable GitHub Pages**
   - Go to Settings > Pages
   - Select "Deploy from a branch"
   - Choose "main" branch and root folder
   - Your site will be live at `https://pheliridge.github.io`

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2E5C6E;      /* Dark Blue */
    --secondary-color: #FF6B35;    /* Orange */
    --accent-color: #F7931E;       /* Yellow */
    /* ... other colors ... */
}
```

### Content
Edit `index.html` to update:
- Organization name and logo
- Mission and vision statements
- Programs and services
- Testimonials and success stories
- Blog posts
- Contact information

### Contact Form Backend
Currently, the form shows a success message. To actually send emails, integrate with:
- **Formspree** (free, no backend needed)
- **EmailJS** (JavaScript-based)
- **Your own backend** (Node.js, Python, etc.)
